    Final Steps¶

    The final step for creating a new event is to record the URL to the event in COMERS OVERVIEW. This URL is used when publishing the event on Ängsbackas website or sent directly to workshopleaders/participants.

    • Open https://iframeang.comers.se/pages/
    • Click Choose button on the new event
    • Copy the URL in the adressbar
    • Open COMERS OVERVIEW
    • Paste the URL in the column LINK
    • Set column STATUS to DONE

    Note

    The URL needs to start with https://iframeang.comers.se. URL's starting with https://adminang.comers.se will not work.
